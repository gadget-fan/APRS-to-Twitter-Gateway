Installation of the server software.

This software depends on ruby, and the twitter gem. Install these before continuing.

The following assumes the use of a debian based distro.

$ apt-get install ruby
$ apt-get install rubygems
$ gem install twitter

Once completed you'll want to clone the software (or just download and extract it from the links in the header).

$ git clone https://github.com/KR0SIV/APRS-to-Twitter-Gateway.git
$ cd APRS-to-Twitter-Gateway

Configure APRS details

The majority of this has been done for you however you need to make a few changes. Starting on line 77 Please replace callsign_here with your callsign, the message on connection and your coordinates should be used, not mine.

  aprs = Aprs.new("second.aprs.net", 20157, "Callsign_here")
  aprs.connect
  aprs.packet("=4148.19N/08056.81W-", "Any_Message_Here")
